## Setup

### Prerequisites

- Python 3.10+
- Docker & Docker Compose
- Poetry

### Installation

1. Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/crypto-backtest-framework.git
cd crypto-backtest-framework
```

2. Install dependencies:
```bash
poetry install
```

3. Start QuestDB:
```bash
docker-compose up -d
```

4. Copy environment file:
```bash
cp .env.example .env
```

5. Verify setup:
```bash
poetry run python -c "import pandas; print('âœ… Setup complete!')"
```
